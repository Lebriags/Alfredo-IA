# 🤵 Alfredo

**Alfredo** é um projeto de estudo desenvolvido em **Python**, com foco em **Inteligência Artificial**, **automação de tarefas** e **controle do computador por comandos de voz**.

O objetivo do projeto é desenvolver um assistente capaz de receber comandos do usuário, interpretá-los e executar ações diretamente no sistema operacional, sem necessidade de interação manual.

Diferente de um chatbot, o Alfredo foi projetado para ser um **executor de tarefas**, não um assistente conversacional.

---

## 🎯 Objetivo

Permitir que o usuário controle o computador utilizando comandos de voz para executar tarefas de forma simples, rápida e automatizada.

Fluxo de funcionamento:

```text
Usuário
   │
   ▼
Comando de voz
   │
   ▼
Interpretação
   │
   ▼
Execução
```

---

## ⚙️ Funcionalidades

O Alfredo foi projetado para ser capaz de:

* Executar comandos por voz.
* Abrir, fechar e controlar aplicativos.
* Manipular arquivos e pastas.
* Navegar na internet.
* Realizar pesquisas na web.
* Abrir páginas e gerenciar abas.
* Preencher campos de texto automaticamente.
* Interagir com botões, menus e outros elementos de páginas web.
* Controlar teclado e mouse.
* Executar comandos básicos no terminal.
* Automatizar tarefas repetitivas no computador.

---

## 🛡️ Segurança

O Alfredo possui uma camada de segurança responsável por validar comandos antes da execução, impedindo operações que violem as regras definidas para o projeto.
